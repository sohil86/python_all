1. The "Source" folder has the source code (RANDOM_NUMBER.c) and the object file (RANDOM_NUMBER.o).

2. The "Include" folder has the header file (HEADER.h).

3. The "Build" folder contains the Makefile.

4. The "Bin" folder contains the executable file.

5. To run the executable on command line, go to bin folder and write ./RANDOM_NUMBER
